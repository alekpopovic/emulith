Target agent: Codex with GPT-5.6  
Prerequisite: Tasks 00–43.

## Required workflow

1. Run the complete baseline suite before edits.
2. Inspect all AWS and Azure handlers, listeners, state migrations, snapshot logic, compatibility reports, Docker files, workflows, and docs.
3. Produce a concise prioritized findings list:
   - release blocker;

## Queue hardening

Verify and fix:

- encoding behavior with official SDK;
- TTL and visibility precision;
- pop receipt uniqueness/rotation;
- stale receipt errors;
- competing consumers;
- clear/delete/reset races;
- queue list pagination;
- message/request limits;
- restart/export/import;
- no collision with AWS SQS.

No long polling, poison queue, server-side auth, or triggers.
