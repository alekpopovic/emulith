# Open-source commitments

The Emulith project commits that:

- the local/CI core requires no account, license key, activation, or entitlement check;
- the core has no forced telemetry;
- supported protocol correctness is not reserved for a closed tier;
- compatibility tests remain public;
- core releases use an OSI-approved open-source license;
- governance changes are public; and
- paid hosting, support, or services may exist without making the local core proprietary.

These are governance commitments, not guarantees beyond the applicable license and contributor rights. Open source permits commercial use and redistribution; AGPL-3.0-or-later imposes source-availability obligations under its terms. This is general project information, not individualized legal advice.
